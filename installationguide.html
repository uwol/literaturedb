literaturedb installation

Introduction

The installation of literaturedb takes four simple steps and just a few minutes. If you want to update an existing installation of literaturedb, please take a look here. Requirements for installation are:

  • PHP 5.0 or higher
  • MySQL 4.1 or higher
  • PHP needs read and write access to the folder "custom/documents". This is not a problem in most PHP configurations.
  • UTF8 has to be enabled in PHP and MySQL for supporting special characters

Installation

In the following the URL http://literaturedb.example.org is used as an example. Please replace it with the URL of your web site you are installing literaturedb on.

  1. Configure: The file custom/systemconfig.php is the central configuration file. It can be edited e.g. with Wordpad or Notepad++. Please enter the connection settings for the MySQL server under $mysqlServer, $mysqlUser, $mysqlPass and $mysqlDb. Normally you can find this information in the configuration menu of your web hoster. Additionally the name of the system, email address of the primary web master (you?), domain name of your literaturedb web site, site URI and admins have to be set.
  2. Upload: Upload all files and folders to the root folder of the web hosting in BINARY mode, so that it can be reached as e.g. http://literaturedb.example.org/index.php.
  3. Create tables: Start install.php by opening e.g. the web page http://literaturedb.example.org/install.php in your browser. All tables should be created in the database properly. Afterwards delete the file install.php.
  4. Register an admin account: Open the site (e.g. http://literaturedb.example.org) in your browser and register a new user. After registering check that the username of the user you just created is also contained in custom/systemconfig.php in the $admins list so that the new user is an administrator. The admin user account is activated automatically. Also a registration notification should have been sent to your email address.

Use

Now all configuration steps are done! Log into the system with the new user account you just registered and upload some documents. Invite some colleagues for registration, activate their accounts on the admin page and share documents with them.

And please don't forget to make backups of the files and database on a regular basis!

Update

For updating just overwrite all installed files and folders (index.php, lib, pages, ...) with files from the updated package EXCEPT the folder custom. DO NOT overwrite the folder custom, because all your document files are placed in there! Then optionally you can check if the new systemconfig.php contains new configuration options and copy those options into your old systemconfig.php in the folder custom.
